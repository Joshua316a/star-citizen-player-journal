# Star Citizen Player Journal

A Python application for tracking and managing your Star Citizen gameplay experience.

## Features

* Track gameplay sessions

* Log missions and activities

* Record earnings and expenses

* Monitor ship inventory

* Track reputation and standings

* Session statistics and analytics

## Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/star-citizen-player-journal.git
cd star-citizen-player-journal

# Create a virtual environment
python -m venv venv

# Activate virtual environment
# On Windows:
venv\Scripts\activate
# On macOS/Linux:
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt
```

## Usage

```bash
python main.py
```
